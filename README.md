# 🎓 StudentManagementAPI

A simple yet powerful **ASP.NET Core Web API** project for managing students.  
This project demonstrates modern backend development practices using **Clean Architecture principles**.

---

## 🚀 Technologies Used

- ASP.NET Core Web API (.NET 8)
- Entity Framework Core
- SQL Server
- Repository Pattern
- DTO (Data Transfer Object)
- AutoMapper
- FluentValidation
- Dependency Injection
- Swagger (OpenAPI)

---

## 📌 Features

- CRUD operations (Create, Read, Update, Delete)
- Layered architecture (Repository Pattern)
- DTO usage for clean data transfer
- Input validation with FluentValidation
- Global exception handling middleware
- RESTful API design
- Swagger API documentation

---

## 🧠 Architecture


Controller → Repository → Database


---

## ⚙️ Setup Instructions

1. Clone the repository:

git clone https://github.com/ismetakpinar/StudentManagementAPI.git


2. Open project in Visual Studio

3. Update connection string in `appsettings.json`

4. Run migrations:

Update-Database


5. Run the project

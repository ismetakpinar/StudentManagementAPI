# Öğrenci Yönetim Sistemi API

ASP.NET Core Web API (.NET 8) ile geliştirilmiş bir öğrenci yönetim sistemidir.

## Teknolojiler
- ASP.NET Core Web API
- Entity Framework Core
- Repository Pattern
- DTO & AutoMapper
- FluentValidation

## Özellikler
- CRUD işlemleri
- Katmanlı mimari
- Global exception handling
- Validation sistemi
